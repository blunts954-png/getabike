## üí° Tips for Best Results

### Midjourney Settings
- Use `--style raw` for photorealistic results
- Adjust `--s` (stylize) between 250-750
- Use `--ar` for aspect ratios:
  - Hero: `--ar 21:9`
  - Products: `--ar 4:3`
  - Avatars: `--ar 1:1`
  - Social: `--ar 1:1`

### Post-Processing
1. Use Topaz Gigapixel AI for upscaling
2. Apply subtle color grading for consistency
3. Compress with TinyPNG or Squoosh
4. Convert to WebP format for web

### Consistency Tips
- Use seed numbers for related images
- Reference previous images with image prompts
- Maintain black/gold color scheme across all assets
- Keep lighting style consistent (dramatic but elegant)
